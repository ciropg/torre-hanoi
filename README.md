## Cómo ejecutar (local)
### Opción 1: abrir en navegador
1. Descarga/Clona el repositorio
2. Abre `index.html` en tu navegador

### Opción 2: VS Code + Live Server
1. Instala la extensión **Live Server**
2. Abre `index.html`
3. Click en **Go Live**

## Cómo jugar
1. Haz clic en una torre para seleccionar el disco superior
2. Haz clic en otra torre para moverlo
3. Regla: **no puedes colocar un disco grande sobre uno más pequeño**

## Deploy
### GitHub Pages
1. Ve a **Settings > Pages**
2. Source: Deploy from a branch
3. Branch: `main` / folder: `/ (root)`
4. Guarda y espera el enlace del sitio
